üî® Cara Build APK

Metode 1: Android Studio (PALING MUDAH) ‚≠ê

Tidak perlu command line, cukup GUI saja

  1. Download Android Studio
    Kunjungi: https://developer.android.com/studio
  2. Extract file pos-appsheet-android.zip
  3. Buka Android Studio ‚Üí Pilih "Open an Existing Project"
  4. Navigate ke folder android yang sudah di-extract
  5. Tunggu Gradle sync selesai (mungkin perlu download dependencies)
  6. Menu: Build ‚Üí Build Bundle(s) / APK(s) ‚Üí Build APK(s)
  7. Klik "locate" di notifikasi untuk menemukan APK

Metode 2: Command Line

Menggunakan npx (sudah tersedia dengan Node.js)

A. Install Prerequisites:

  • Node.js: nodejs.org
  • JDK 17: adoptium.net

B. Set Environment Variables (Windows):

set JAVA_HOME=C:\Program Files\Java\jdk-17
set ANDROID_HOME=C:\Users\%USERNAME%\AppData\Local\Android\Sdk
set PATH=%PATH%;%JAVA_HOME%\bin;%ANDROID_HOME%\platform-tools

C. Set Environment Variables (Mac/Linux):

export JAVA_HOME=/usr/lib/jvm/java-17-openjdk
export ANDROID_HOME=$HOME/Android/Sdk
export PATH=$PATH:$JAVA_HOME/bin:$ANDROID_HOME/platform-tools

D. Build APK:

cd android
./gradlew assembleDebug    # Mac/Linux
gradlew.bat assembleDebug  # Windows

Metode 3: PWA (TANPA APK) üåê

  1. Buka aplikasi di Chrome (Android)

üì± Install APK di Android

  1. Transfer file APK ke HP (USB/Email/Cloud)
  2. Buka Settings ‚Üí Security ‚Üí Enable "Unknown Sources"
  3. Buka File Manager ‚Üí Tap file APK
  4. Tap Install

üîß Troubleshooting

Error: JAVA_HOME not found
Install JDK 17 dari adoptium.net, lalu set environment variable
Error: ANDROID_HOME not found
Install Android Studio, SDK akan terinstall otomatis
Error: Gradle permission denied
chmod +x android/gradlew (Mac/Linux only)
